# Linux Mint Cleaner (GUI)

A safe, GUI-based system cleaner for Linux Mint, with:

- Cache cleanup
- Trash cleanup
- Thumbnail cleanup
- Flatpak leftover removal
- Timeshift snapshot pruning (keeps newest snapshot)
- Logging
- GUI Zenity interface
- Desktop launcher support

## Installation
There are currently **2** ways to Install Linux Mint Cleaner.
### 1: Using The Debian Package (.deb)
> Each Release comes with a Pre-built Debian Package, you can execute it with the Default Debian Package Installer.
> **IF Your Default Installer Fails to work:**
> Open up a terminal in the same repository you have the Debian Package from and write ```sudo dpkg -i linux-mint-cleaner.deb```
> you may be prompted to put in your system password if you have not set up via gnome to bypass this.
### 2: Using the Install.sh File in the source Code
> In the source code is an install.sh file. It will Handle all the installing for you and you won't need to do anything.

## Uninstallation:
There is currently only **1** Way to Uninstall Linux Mint Cleaner.
### 1: Using the Uninstall.sh File in the source code
> In the source code is an uninstall.sh file. It will Handle all the uninstalling for you and you won't need to do anything.
