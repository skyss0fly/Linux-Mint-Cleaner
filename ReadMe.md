# Linux Mint Cleaner (GUI)

A safe, GUI-based system cleaner for Linux Mint, with:

- Cache cleanup
- Trash cleanup
- Thumbnail cleanup
- Flatpak leftover removal
- Timeshift snapshot pruning (keeps newest snapshot)
- Logging
- GUI Zenity interface
- Desktop launcher support

## Installation

Clone the repo:
```
git clone https://github.com/skyss0fly/mint-cleaner.git

cd mint-cleaner
sudo ./install.sh
```


## Running

Launch it from the Mint Menu under **System Tools**, or run:

```mint-cleaner```



## Uninstall

```sudo rm /usr/local/bin/mint-cleaner
sudo rm /usr/local/share/applications/mint-cleaner.desktop```
